# nickel-todo

A toto application for Nickel on Rust

[![Build Status](https://travis-ci.org/cncgl/nickel-todo.svg?branch=master)](https://travis-ci.org/cncgl/nickel-todo)

## Install
```
$ cargo build --verbose
```

## Usage
debug build
```
$ cargo run --verbose
```

release build
```
$ cargo run --release --verbose
```

## Test
```
$ cargo test --verbose
```
